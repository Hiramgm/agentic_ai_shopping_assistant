Agentic AI Shopping Assistant - Master Project Plan

Project Goal

Build a production-inspired, category-agnostic AI shopping agent demonstrating Agentic AI, RAG, hybrid retrieval, knowledge graphs, recommendations, memory, explainability, and provider abstraction.

Core User Journey

  • Search products using natural language
  • Apply conversational filters
  • Compare products
  • Analyze reviews
  • Receive personalized recommendations
  • Ask follow-up and explanation questions
